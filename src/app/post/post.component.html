Ode to Binary Search Trees

1. Introduction

As many computer science students are aware, Binary Search Trees (BST) is one of the very first data structures to be encountered in the algorithms module. It is an effective starting point as it provides the student with the possibility to learn about trees in general, tree traversal and of course the fundamentals of the BST itself which gracefully guides us towards other types of trees.
